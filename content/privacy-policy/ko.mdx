# 개인정보 보호정책

**최종 업데이트:** 2025년 1월 27일

## 우리가 하는 일

수면 계산기는 수면 일정을 최적화하고 수면 품질을 개선하는 데 도움이 되는 전문 수면 주기 계산 도구를 제공합니다. 우리는 사용자로부터 개인 정보를 직접 수집, 저장 또는 처리하지 않습니다.

## 분석

우리는 웹사이트 사용을 이해하기 위해 다음과 같은 타사 분석 서비스를 사용합니다:

- **Ahrefs Analytics** - 익명 웹사이트 성능 데이터 처리
- **Google Analytics** - 익명 사용 데이터 처리(페이지 조회수, 브라우저 유형, IP 주소 기반 대략적인 위치)

**법적 근거:** 웹사이트 성능 및 사용자 경험 분석에 대한 정당한 이익.

**데이터 전송:** 이러한 서비스는 미국 및 기타 국가로 데이터를 전송할 수 있습니다.

수집된 데이터는 집계되고 가명화됩니다. 우리는 개별 사용자 데이터에 액세스할 수 없습니다.

## 귀하의 권리 (GDPR)

우리는 개인 데이터를 직접 수집하지 않지만, 분석 파트너는 개인 데이터로 간주될 수 있는 일부 데이터(예: IP 주소)를 처리할 수 있습니다.

EU 거주자는 이 데이터 처리에 관한 권리가 있습니다. 귀하의 권리 또는 데이터 처리에 대한 질문은 이메일로 문의하세요.

## 쿠키

분석 파트너는 귀하의 기기에 쿠키를 배치할 수 있습니다. 다음 방법으로 추적을 제어할 수 있습니다:

- 브라우저 설정에서 쿠키 비활성화
- 광고 차단기 또는 개인정보 보호 확장 프로그램 사용
- 브라우저에서 "추적 안 함" 활성화
